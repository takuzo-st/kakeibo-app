# 💰 Kakeibo App（家計簿アプリ）

Spring Bootで作る「実践型ハンズオン教材」です。  
4週間で**家計簿アプリ + AI機能**まで作り切ります。

---

## 🚀 できること

- 支出の登録・一覧表示（CRUD）
- 月別・カテゴリ別の集計
- グラフで可視化（Chart.js）
- AIによる自動カテゴリ分類（OpenAI / Claude）
